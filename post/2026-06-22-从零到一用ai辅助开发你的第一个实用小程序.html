实战：开发一个桌面文件整理工具

目标： 写一个Python脚本，把桌面上杂乱的图片、文档、压缩包自动归类到对应文件夹。

第一步：需求拆解（5分钟）

打开Claude，输入：

我是编程完全新手，想写一个Python脚本，功能：
1. 扫描桌面所有文件
2. 按扩展名分类（.jpg→图片，.docx→文档，.zip→压缩包）
3. 自动创建文件夹并移动文件
4. 要能处理重名文件（加序号）
请给我完整代码，并逐行解释

Claude会立刻给出类似这样的代码：

import os
import shutil
from pathlib import Path

# 获取桌面路径
desktop = Path.home() / "Desktop"

# 定义分类规则
type_map = {
    "图片": [".jpg", ".jpeg", ".png", ".gif"],
    "文档": [".pdf", ".docx", ".txt", ".xlsx"],
    "压缩包": [".zip", ".rar", ".7z"],
    "视频": [".mp4", ".avi", ".mkv"]
}

# 遍历桌面文件
for file in desktop.iterdir():
    if file.is_file():
        # 获取文件后缀
        ext = file.suffix.lower()
        # 找到匹配的分类
        for category, extensions in type_map.items():
            if ext in extensions:
                # 创建分类文件夹
                target_dir = desktop / category
                target_dir.mkdir(exist_ok=True)

                # 处理重名：加数字后缀
                dest = target_dir / file.name
                counter = 1
                while dest.exists():
                    stem = file.stem
                    dest = target_dir / f"{stem}_{counter}{ext}"
                    counter += 1

                # 移动文件
                shutil.move(str(file), str(dest))
                print(f"已移动: {file.name} → {category}")
                break

print("整理完成！")

第二步：运行与调试（15分钟）

  1. 安装Python：去python.org下载安装，记得勾选“Add Python to PATH”

  3. 粘贴代码：把Claude生成的代码复制进去

如果报错，直接把错误信息复制给Claude，让它帮你修。比如：

Claude会告诉你关闭占用文件的程序，或者用管理员权限运行。

进阶技巧：如何让AI写出更好的代码

不是所有AI输出都靠谱。三个原则让你事半功倍：

  1. 说人话，但要有细节：不要只说“写个程序”，要说“用Python 3.10，处理桌面路径用Pathlib，遇到同名文件加数字后缀”
  2. 分步提问：先让AI写核心逻辑，再一步步加功能，比一次提所有需求更准确
  3. 让它解释：如果看不懂代码，直接说“请用比喻解释这段代码”，AI会秒变老师

真实案例：一位博主用同样方法，让Claude生成了一个自动备份微信聊天记录到云盘的脚本，全程没写一行代码，只用了4轮对话。
